Ask Lovable prompt to store the credentials needed for Freemius Integration. You can get it from the [Developer Dashboard](https://freemius.com/help/documentation/saas-sdk/js-sdk/installation/#retrieving-keys-from-the-developer-dashboard).

- head over to the Developer Dashboard and navigate to the Settings page for your product and selec the API & Keys tab.
- Now scroll down to the Usage examples, copy the code snippet written in the .env format.

---

We are going to have Freemius Checkout and Licensing Integration for our SaaS. Store the following credentials in the application environment variables.:

FREEMIUS_PRODUCT_ID=""
FREEMIUS_API_KEY=""
FREEMIUS_SECRET_KEY=""
FREEMIUS_PUBLIC_KEY=""

Store the SECRET_KEY and API_KEY in a secure way. The Product ID and Public Key needs to be available in the front-end.

---

The platform might ask you for the values again, please enter it in their UI.
